# 🚀 AI Resume Analyzer

An **AI-powered Generative AI project** that analyzes resumes using LLMs and provides smart career insights.

## ✨ Features
✅ Skill Extraction  
✅ Missing Skill Detection  
✅ ATS Score Suggestions  
✅ Interview Question Generation  
✅ Resume Improvement Tips  

---

## 🛠 Tech Stack
- 🐍 Python  
- 🎨 Streamlit  
- ⚡ Groq API  
- 🤖 Llama LLM  

---

## 📌 How It Works
📄 Upload Resume  
⬇️  
🧠 AI analyzes skills and profile  
⬇️  
📊 Get ATS suggestions, missing skills, and interview questions  

---

## ▶️ Run Locally

Clone repository:

```bash
git clone https://github.com/yourusername/genai-resume-analyzer.git
cd genai-resume-analyzer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run app:

```bash
streamlit run app.py
```
